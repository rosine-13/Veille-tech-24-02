# Veille Technologique - 24 Février 2026

Ce projet présente ma veille sur les fondamentaux du numérique et l'intelligence artificielle.

## Fonctionnalités DOM
Pour ce rendu, j'ai utilisé le **DOM** pour créer des interactions :
- **Bouton Mode Sombre :** Utilisation de `querySelector` et `classList.toggle` pour changer le thème.
- **Interactivité Cartes :** Utilisation de `addEventListener` pour détecter les clics sur les sections (ex: la carte IA).
- **Modification dynamique :** Changement de style en temps réel via JavaScript.

## Thèmes abordés
1. Électricité
2. Ordinateur
3. Logiciel
4. Internet
5. Smartphone
6. IA
7. Super Ordinateur
